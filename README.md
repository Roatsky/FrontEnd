Ini adalah front end untuk website saya buat final project web development pekan ristek
Saya disini akan membuat website menggunakan next.js yang mengandung manajemen matkul, manajemen tugas, dan tracker tugas
Tentu feature opsional kalkualtor nilai dan bolos tracker ingin dibuat, tapi itu tergantung apakah berhasil dikejar

Untuk struktur, saya ngebayang strukturnya seperti berikut:
/app (folder buat websitenya)
  /page (file-file buat gambar webnya)
  /feature (fungsi-fungsi untuk setiap fitur web)
  /asset(file-file seperti link atau gambar untuk memperbagus web)
/public (folder public langsung dari next.js)
